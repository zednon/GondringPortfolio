About The Project

UI Style Guild

I spend so much time teaching I wanted to update my portfolio with a style guild project.

Landing Page

The key part to any good website is the landing page, I put a place holder graphic in a call to action button and header to grab the users attention. The navgation bar has a blue dot to let the user know what page they are on.

Guild Lines

Setting up rules for a site is very import, I started with a mood board and found colors that would would for a palette for this site. I really found this img very pretty and wanted to focus on it. I used the color picker tool in fimga and pulled colors out of the img and used that for the palette for this projected.

Mood Board

Here is my mood board for this project with my colors hex codes at the side.

Accessibility Testing

Accessibility testing the Palette right away in the project before designing to ensure the project is always accessible regarding to the colors used.

Components

Using Atom Design I created states for these Assents which I used threw out this project

Text and Button

I started with created the rule set for the text so I could use the text with changes I wanted when a mouse would enter and leave. The button I proved states for Normal, Pressed, and Focused.

NavBar

I made a NavBar, using the text from Text and Buttons states, when a mouse enters the text bolds or changes colors the drop down menu has a onclick function.

Cards

Even though cards are components, I wanted to put them in there own area since they are often a mix of many elements. Using atom design I'm able to make this cards ones then just repeat them threw out the page
